# Figma Plant UI

A modern responsive plant landing page built using React and Vite.

## Features

- Responsive Navbar
- Hero Section
- Trending Plants
- Top Selling Plants
- Customer Reviews
- O2 Plants Section
- Newsletter Section
- Footer

## Tech Stack

- React
- Vite
- CSS
- React Icons

## Installation

Clone the repository

```bash
git clone https://github.com/praveenagurubalan25/figma-plant-ui.git
```

Go to the project

```bash
cd figma-plant-ui
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
